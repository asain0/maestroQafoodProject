# 📱 Maestro Mobile Tests (Projeto de Estudos)

Este repositório tem como objetivo o aprendizado e experimentação com o **Maestro Framework** para automação de testes em aplicações mobile.

## 🎯 Objetivo

Criar um ambiente de estudo prático para:
- Aprender a estrutura e funcionamento do Maestro
- Automatizar fluxos de testes em um app mobile
- Experimentar boas práticas de automação
- Integrar testes com pipelines de CI futuramente

## 🧪 Sobre o Projeto

Este projeto **não está vinculado a nenhum aplicativo real**.  
Os testes aqui desenvolvidos serão aplicados em um **app fictício**, criado apenas para fins de aprendizado e validação de cenários.

## 🚀 Tecnologias e Ferramentas

- Maestro Framework
- Git e GitHub
- (Futuro) CI com GitHub Actions
- (Opcional) Android Emulator / iOS Simulator
